Assignment : Membuat halaman login

Panduan Aplikasi :

1. Download folder tugas
2. Buka pada code pada Visual Studio Code
3. Buka terminal dan pastikan sudah berada didalam folder tugas
4. Jalankan program dengan mengetikkan command pada terminal dengan perintah :
   go run tugaslogin.go
5. Tekan enter dan output akan keluar. 
Output berupa baris untuk mengisikan username dan password.
Jika password yang diinputkan sesuai dengan yang sudah ditentukan maka akan keluar text berhasil masuk.
Jika password yang diinputkan tidak sesuai dengan yang sudah ditentukan maka akan keluar text Username/Password yang digunakan salah.
